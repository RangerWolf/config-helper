# Pip-Extension



## 简介

简称 `pip-ex`

希望能扩展pip的一些功能。  

目前唯一的功能就是针对pip的源做一些修改。 



* list

  列出所有的源。 

* change

  * aliyun
  * ustc
  * tsinghua

  有上面3个合法的目标值。
